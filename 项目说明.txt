项目名称：彩云之南 - 云南旅游专题网站 (Yunnan Travel Guide)
1. 项目概述 (Project Overview)
项目简介：
本项目是一个专注于介绍中国云南省旅游资源的静态网站。网站以“彩云之南”为主题，通过精美的视觉设计和详实的内容，旨在为用户提供一个沉浸式、信息丰富的在线云南旅游向导。网站内容涵盖了云南的四大核心旅游主题：美食、景点、人文和文化，并通过多种交互功能提升用户体验。
目标用户：
对云南旅游感兴趣的潜在游客、文化爱好者、地理爱好者以及寻求旅行灵感的网络用户。
项目愿景：
旨在打造一个灵动美观的旅游介绍的网页。通过现代化的网页设计和技术，展现一个兼具自然之美与人文之厚的七彩云南，为家乡宣传出一份力。

2. 网站结构与功能模块 (Site Structure & Modules)
本网站采用多页面静态网站结构，主要包含以下页面和功能模块：
	
•	主页 (index.html)

全屏英雄轮播图: 以全屏、带文字和行动号召（CTA）按钮的轮播图作为开场，分别展示四大主题（美食、景点、人文、文化），提供极强的视觉冲击力，并引导用户进入相应的主题子页面。
交互式地图: 采用图片热区的形式，展示云南省16个地州市的行政区划图。用户鼠标悬停时有高亮效果，点击可跳转至对应的百度百科页面，提供地理探索的趣味性。
核心分类导航: 以图标和文字结合的卡片形式，清晰地展示网站的四个核心内容板块，方便用户快速导航。
•	关于我们 (about.html)

交错式图文卡片: 采用左右图文交错的“杂志式”布局，分别介绍四大主题的核心理念，版式富有节奏感，阅读体验舒适。
	粘性垂直导航: 右侧的垂直导航栏在页面滚动时会“粘性”固定，方便用户随时在不同模块间切换，提升了页面的可用性。
•	主题子页面 (theme_*.html x 4)
	
	通用模板: 四个子页面（美食、景点、人文、文化）均采用统一的、灵感源于“挪威风格”的沉浸式专题布局，保证了网站风格的一致性。
	英雄区域: 每个子页面都以一个与主题相关的全屏高清图片开篇，配有主副标题，营造强烈的代入感。
	叙事性内容流: 采用“宽窄容器穿插”和“全宽大图”的布局，引导用户像阅读一篇精美的专题文章一样浏览内容。
	左侧大卡片 + 右侧垂直导航: 在内容详情区域，采用两栏布局。左侧以一个带有圆角和阴影的“大卡片”承载详细的图文列表；右侧则提供一个便捷的垂直导航，有效利用了宽屏空间并提升了导航效率。
	底部推荐卡片: 每个子页面底部都有一个两行两列的推荐卡片区域，用于高亮本主题下的核心体验。
•	联系我们 (contact.html)
	
	多功能表单: 提供一个包含姓名、邮箱、电话、多选主题和留言功能的联系表单。
	前端验证: 使用 JavaScript 对邮箱格式（必须包含@和.）和电话号码（必须为11位纯数字）进行实时验证，并提供清晰的错误提示，提升了用户输入的准确性和体验。
	自定义多选下拉框: “主题”字段采用自定义的、支持多选的下拉框组件，用户体验优于原生下拉框。
•	全站通用功能
	
	响应式设计: 网站所有页面均采用响应式布局，能够较良好地适配桌面、平板和手机等不同尺寸的设备（仍需改进）。
	页眉与页脚: 统一的页眉包含网站标题、主导航和全站搜索功能；统一的页脚包含版权信息。

3. 技术实现与创新点 (Technology & Innovations)
本项目虽然是一个静态网站，但在实现过程中运用了多种现代前端技术和设计理念，以达到媲美动态网站的交互体验和视觉效果。
	
•	HTML5 & CSS3:
	
	Flexbox & Grid 布局: 广泛应用于页眉、页脚、两栏布局和卡片网格中，实现了复杂布局的现代化、响应式构建。
	CSS 伪元素 (::before, ::after): 用于创建英雄区域的半透明遮罩、子导航栏的动态下划线（后改为圆角矩形背景），实现了内容与样式的分离，代码更具可维护性。
	CSS 动画与过渡 (@keyframes, transition): 为轮播图的淡入淡出、文字入场、按钮悬停、搜索框聚焦等交互添加了平滑的动画效果，极大地提升了页面的精致感和用户体验。
	CSS 变量与模块化: 将 CSS 拆分为 base.css, layout.css, components.css, theme-page.css, responsive.css 五个文件，实现了样式的模块化管理，提高了代码的可读性和可维护性。
	object-fit & aspect-ratio: 用于处理图片，确保图片在不同尺寸的容器中都能以正确的比例显示，避免拉伸变形。
•	JavaScript (原生 ES6+):
	
	DOM 操作: 动态创建轮播图的小圆点、生成搜索结果列表等。
	事件监听: 实现了对轮播图按钮点击、表单提交、搜索等用户行为的响应。
	全功能轮播图: 从零开始构建了一个支持自动播放、手动切换、小圆点导航的全屏淡入淡出轮播图，逻辑清晰，性能良好。
	前端表单验证: 实现了对邮箱和电话号码的正则表达式验证，提供了即时的用户反馈。
	客户端搜索功能:
	
	静态索引: 通过手动维护 search-data.js 文件，构建了一个完整的客户端搜索索引。
	精准定位: 创新性地在索引中加入了锚点链接（#id），并配合 JavaScript 实现了搜索结果不仅能跳转到目标页面，还能平滑滚动到页面内的具体条目。
	视觉高亮: 跳转后，目标条目会有一个临时的、带动画的高亮效果，明确地向用户指示搜索结果的位置，极大地优化了搜索体验。
•	设计与用户体验 (UX):
	
        移动优先与响应式设计: 确保了网站在所有设备上都有较好的浏览体验（仍需优化）。
	引入 Web Fonts: 使用 Google Fonts 提供的 Noto Serif SC (标题) 和 Source Han Sans CN (正文)，显著提升了网站的排版质量和艺术感。
	色彩与留白: 借鉴 VisitNorway.com 的设计风格，大胆使用留白，并为内容区引入了柔和的浅绿色背景，营造了高级、舒适的视觉氛围。
	组件化思想: 将网站拆分为导航、卡片、表单等可复用的组件，分别进行设计和编码，提高了开发效率和代码质量。

4. 素材来源 (Asset Sources)
	
•	图片素材:
	
	本项目中使用的所有风景、美食、人文、文化相关的图片，其来源均为百度百科对应的词条。图片仅作为本项目学习与开发过程中的占位符和视觉示例使用，不用于任何商业用途。
	图片列表包括但不限于：food-.jpg, scenery-.jpg, human-.jpg, culture-.jpg 等。
•	图标库:
	
	使用了 Font Awesome (v6.x) 图标库，通过 CDN 引入。主要用于 about.html 和各子页面右侧的垂直导航栏，以及首页的分类导航中，提供了清晰、统一的矢量图标。
•	字体:
	
	使用了 Google Fonts 提供的免费开源字体，通过 CSS @import 方式引入，包括 Noto Serif SC 和 Source Han Sans CN。
